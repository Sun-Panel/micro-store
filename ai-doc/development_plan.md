## 开发规范

### 后端开发
1. 在 `service/models/` 创建模型（已完成）
2. 在 `service/api/api_v1/` 创建 API 控制器
3. 在 `service/router/` 注册路由
4. 在 `service/biz/` 处理复杂业务逻辑（如需要）

### 前端开发
1. 在 `src/api/` 创建 API 接口
2. 在 `src/typings/` 定义类型
3. 在 `src/views/` 创建页面
4. 在 `src/router/` 注册路由
5. 在 `src/locales/` 添加多语言文本

### Git 提交规范
```
feat: 新功能
fix: 修复bug
docs: 文档更新
style: 代码格式调整
refactor: 重构
test: 测试相关
chore: 构建/工具相关
```
